Sublime Text 3-JavaScript编译器

发表于 2017-07-29 | 分类于 sublime-Text-3使用笔记

1. 前奏

在学习js的过程中，使用了传说中的神编辑器—-sublime Text，以前配置了一部分，今天又配置了一些方便使用的插件，这个编辑器真是666啊，明明有专门编辑JavaScript的我不用，偏偏来这找罪受，没办法，谁让咱爱折腾呢。

使用js的时候，发现调试代码没有办法，sublime自带的编译系统不支持js，在网上一通乱找，解决了。

2. NodeJS安装

首先，我们需要安装NodeJS

下载后一路继续安装成功后，nodeJS默认会添加到环境变量Path里面，这方便了我们以后的使用。

3. 在编译系统里面添加JavaScript编译器

这一步默认你已经下载并安装好了sublime Text了，并且熟悉里面的一些插件的操作。
我们选择 工具—>编译系统—>新建编译系统（此时你会看到默认的编译系统还是支持很多语言的，就是没有JavaScript）, 然后会出来一个未保存的文件
