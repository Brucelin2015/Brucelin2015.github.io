自定义log

今天在学习swift的时候，发现 print() 函数打印出来的信息真的很少，￼

这对于我们调试很不方便，看视频的时候教了一个方法来自定义log，记下来，以后省的忘记了。

上面的代码中有三处值得注意的地方

  1. log\, 这个T，指的是泛型。T作为一个占位类型名来代替实际类型名，(如Int，String，Double)。占位类型名没有指明T必须是什么类型，简单点说，你准备输出的message是什么类型，T就是什么类型
  2. 函数里面 fileName: String = #file 指的是函数的默认参数，一旦你给一个参数赋值了，每次调用函数的时候默认就是该值，这里我们设置了三个默认值，调用的时候, 就不必再给后面的参数传值了
  3. #file、#function， 表示调用这个方法的当前文件和在哪个函数里面调用的， #line，这一句表示方法在哪一行执行，就把该行显示出来

全局输出

如果你想要全局输出的话，把该函数直接放到AppDelegate的最下面，不让它在AppDelegate类里面就是全局函数了
