sublime Text 3 配置Python3编译环境

发表于 2017-07-29 | 分类于 python3 ， sublime-Text-3

最近在学习python3，使用sublime Text编辑器，发现莫名的爽，编码界面看着很舒服，代码的适应也很好，更有非常多的插件，可以满足几乎所有语言的学习。

开始配置(windows环境)

使用sublime Text，肯定是需要 Package Control ，这是玩sublime Text必不可少的，你肯定已经安装并指导怎么使用了，不会的话百度一下。

点击， 工具—->编译系统—->新建编译系统,粘贴下面代码，保存到默认路径，设置名字为Python3(随意取)

{
    "encoding": "utf-8",  
    "working_dir": "$file_path",  
    "shell_cmd": "C:\\Users\\chenglong\\AppData\\Local\\Programs\\Python\\Python36\\python.exe -u \"$file\"",  
    "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",  
    "selector": "source.python"
}

上面的shell_cmd字段填的是你下载的python的执行程序的绝对地址，找到你的python路径，复制，粘贴到sublime text的时候，需要转义，就是在 \ 前面再一个 \, 否则会报错

保存以后，选中刚刚你命名的python编译系统
